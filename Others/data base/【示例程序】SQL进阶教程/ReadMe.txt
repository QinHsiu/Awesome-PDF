●关于示例程序

示例程序以程序所在页码数为名命名。例如，第11页记述的1-1节的示例程序保存的位置和文件名如下所示。

:\
└─Ch1(1-1)
        └─Sample
            └─p11_1.txt

创建示例用表的SQL语句以“表名”的方式命名。例如，创建1-1节中的表CourseMaster的SQL语句保存的位置和文件名如下所示。

:\
└─Ch1(1-1)
        └─CreateTable
            └─CourseMaster.txt
